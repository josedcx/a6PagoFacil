# a6PagoFacil
Prueba técnica pago fácil.

Este proyecto es generado con fines de evaluación en Angular 6, para la empresa Pagó Fácil.

## Server
Ejecute `npm install` para iniciar la descarga e instalación de las dependencias.
 
Ejecute `ng serve` para iniciar el servidor dev. Vaya a `http://localhost: 4200/`. 

## Dato importante

El requerimiento no se si fue bien descrito con respecto al funcionamiento del WS, ya que en las indicaciones solicitan que
se muestre la lista antes de hacer login y en este caso haciendo pruebas con el postman me percaté que el funcionamiento
de los WS es que tenga un login primero para generar un Token.
